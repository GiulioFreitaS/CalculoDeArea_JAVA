# 📐 Calculadora de Áreas em Java

Este projeto é um programa simples em **Java** que calcula a área de algumas formas geométricas a partir da escolha do usuário.

O usuário escolhe qual forma deseja calcular e depois informa as medidas necessárias.

## 🔢 Formas disponíveis

* Quadrado
* Retângulo
* Círculo

## 🧩 Estrutura do Projeto

O projeto foi dividido em três classes principais para organizar melhor o código:

### Pedido

Responsável por interagir com o usuário.
Mostra o menu de opções, recebe os dados digitados e chama o cálculo correspondente.

### CalculoGeometria

Responsável por realizar os cálculos das áreas das figuras geométricas.

### Saida

Responsável por exibir o resultado do cálculo na tela.

## ⚙️ Funcionamento

1. O programa inicia na classe **Main**.
2. A classe **Pedido** mostra um menu com as opções de formas geométricas.
3. O usuário escolhe uma opção e informa as medidas necessárias.
4. A classe **CalculoGeometria** realiza o cálculo da área.
5. A classe **Saida** exibe o resultado.

## ▶️ Exemplo de uso

```
Escolha a área:
1 - Quadrado
2 - Retângulo
3 - Círculo

Digite a opção: 1
Digite o lado: 5

A área é: 25
```
